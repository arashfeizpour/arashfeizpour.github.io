Have you ever confronted an equation which can not be solved analytically for a variable? Sure you have . . One of the best ways to find the answer in these equations is to draw both sides of the equation and then see where they intersect!
In this program we want to solve this equation : sqrt(l - y**2)/y = tan(y). As you can see it is extremely challenging to solve this for y analytically. So we simply take one side of the equation and draw it, also draw the other side in the same figure. Now, noting a simple phenomenon we can find the intersection of these two diagrams. This phenomenon is : sign = np.sign(leftSide - rightSide ) - 1 , where the subtraction of the values of one side from the other changes sign!

OUTPUTs :
the intersection point is : 0.934019251769
